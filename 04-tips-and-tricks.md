# 💡 Tips & Tricks Guide

Power-user techniques to help you save time and work more efficiently.

---

## ⌨️ Keyboard Shortcuts

Master these to avoid reaching for your mouse constantly:

### Navigation
| Shortcut | Action |
|----------|--------|
| `G + H` | Go to Home |
| `G + S` | Go to Settings |
| `G + N` | Go to Notifications |
| `Esc` | Close any open panel or dialog |

### Editing
| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + Z` | Undo last action |
| `Ctrl/Cmd + Shift + Z` | Redo |
| `Ctrl/Cmd + A` | Select all |
| `Ctrl/Cmd + D` | Duplicate selected item |

### View
| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + +` | Zoom in |
| `Ctrl/Cmd + -` | Zoom out |
| `Ctrl/Cmd + 0` | Reset zoom |

## 🔍 Search Like a Pro

Go beyond basic keyword search:

- **Exact phrase:** Put quotes around it → `"final report"`
- **Exclude a word:** Use a minus sign → `report -draft`
- **Search by field:** Use prefixes → `title: budget`, `tag: urgent`, `author: Amit`
- **Date ranges:** → `date: 2025-03-01..2025-03-31`

**Example:**
```
title: "Q1 review" tag: finance -archived
```

## 📌 Pinning & Favouriting

- **Pin** items to keep them at the top of your list
- **Star/Favourite** items you access often for one-click access from the sidebar
- Right-click any item to see pin and favourite options

---

## 🗂️ Organising with Tags

Tags are the fastest way to group related items across different folders:

1. Open any item and click **Add Tag**
2. Type a tag name and press Enter
3. Use the same tag on related items
4. Filter by tag in search to see everything grouped

> 💡 Use consistent naming: e.g., `q1-2025`, `client-abc`, `urgent`

---

## 📋 Using Templates

Save yourself setup time by using templates:

1. Go to **+ New → Use Template**
2. Pick a template from the library
3. Customise the pre-filled fields
4. Save as a new item

**Create your own template:**
1. Set up an item exactly how you like it
2. Click **More Options → Save as Template**
3. Give it a name and save

## 🔔 Smart Notifications

Avoid notification overload:

- Turn off **Email** notifications for minor updates — use **In-app** instead
- Set a **Daily Digest** for low-priority items so they arrive once a day
- Use **@mentions** to alert specific people instead of notifying everyone

---

## 📊 Bulk Actions

Save time when managing multiple items at once:

1. Tick the checkbox on multiple items
2. A **bulk action bar** appears at the bottom
3. Choose: **Delete**, **Export**, **Tag**, **Move**, or **Archive**

> 💡 Use `Shift + Click` to select a range of items quickly.

---

## 🔗 Sharing Links Directly

Every item has a permanent, shareable link:

1. Open the item
2. Click **Copy Link** (or use `Ctrl/Cmd + Shift + C`)
3. Share the link — recipients with access will land directly on the item

---

## 🕓 Version History

Made a mistake? Roll back to an earlier version:

1. Open the item
2. Click **More Options → Version History**
3. Browse previous versions by date and author
4. Click **Restore** on the version you want

---

## 🌙 Dark Mode Shortcut

Toggle dark mode instantly without going to Settings:

- Click your **avatar → Toggle Dark Mode**
- Or press `Ctrl/Cmd + Shift + D` (if enabled in your keyboard settings)
