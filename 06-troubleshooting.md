# 🆘 Troubleshooting Guide

Step-by-step fixes for the most common issues users encounter.

---

## 🔴 Can't Log In

**Symptoms:** Login page shows an error, or you're stuck in a loop.

**Steps to fix:**
1. Double-check your email and password (watch for CAPS LOCK)
2. Click **Forgot Password** to reset if unsure
3. Clear your browser cache: `Ctrl/Cmd + Shift + Delete → Clear cache → Reload`
4. Try a different browser or incognito/private window
5. Disable browser extensions (some ad blockers interfere)
6. If the problem persists, contact support with a screenshot of the error

---

## 🟡 Page Not Loading or Blank Screen

**Symptoms:** White screen, spinner that never stops, or partial page load.

**Steps to fix:**
1. Refresh the page (`F5` or `Ctrl/Cmd + R`)
2. Check your internet connection
3. Clear cache and cookies: `Ctrl/Cmd + Shift + Delete`
4. Try a hard refresh: `Ctrl + Shift + R` (Windows) / `Cmd + Shift + R` (Mac)
5. Disable browser extensions one by one to identify a conflict
6. Try a different browser

---

## 🟡 Changes Not Saving

**Symptoms:** Edits disappear after refreshing or navigating away.

**Steps to fix:**
1. Look for a **Save** or **Save Changes** button — not all fields auto-save
2. Check for a warning banner at the top of the page (e.g., "You have unsaved changes")
3. Ensure your internet connection is active
4. Try saving using the keyboard shortcut `Ctrl/Cmd + S`
5. If still failing, copy your work, refresh, and re-paste

---

## 🟡 File Upload Failing

**Symptoms:** Upload gets stuck, shows an error, or disappears.

**Steps to fix:**

| Check | Details |
|-------|---------|
| File size | Must be under **25 MB** |
| File type | Must be a supported format (PDF, DOCX, PNG, etc.) |
| Connection | Weak internet can cause upload failures |
| Browser | Try Chrome or Firefox if the issue persists |

1. Check the file size and type first
2. Reload the page and try again
3. If uploading multiple files, try one at a time

---

## 🟡 Search Returning No Results

**Symptoms:** Search finds nothing even when you know the item exists.

**Steps to fix:**
1. Check your spelling
2. Try a shorter or different keyword
3. Remove any active filters that may be hiding results
4. Check if the item is in the **Trash** (it won't show in regular search)
5. If the item was shared with you, make sure the owner hasn't removed access

---

## 🟡 Not Receiving Notifications

**Symptoms:** You're missing alerts for updates, mentions, or messages.

**Steps to fix:**
1. Go to **Settings → Notifications** and confirm notifications are turned on
2. Check your **spam/junk folder** for email notifications
3. On mobile, make sure push notifications are allowed for the app in your device settings
4. If using a browser, make sure you haven't blocked browser notifications from the site

---

## 🟡 Shared Items Inaccessible to Others

**Symptoms:** A colleague says they can't see something you shared.

**Steps to fix:**
1. Confirm you used their correct email address
2. Ask them to check if they received an invite email (check spam too)
3. Open the item and re-check **Share settings** — make sure the correct permission level is set
4. If the invite expired, re-send it from **Share → Manage Access**

---

## 🔴 Account Locked or Suspended

**Symptoms:** You see a message saying your account has been locked or suspended.

**Steps to fix:**
1. Check your email for a notice from the system about the reason
2. If locked due to too many login attempts, wait 15 minutes then try again
3. If suspended for a policy reason, contact support directly

---

## 🔴 Data Appears Missing or Incorrect

**Symptoms:** Items, folders, or content you created are gone or showing wrong info.

**Steps to fix:**
1. Check the **Trash** — items are kept there for 30 days before permanent deletion
2. Check **Version History** on the item (`More Options → Version History`) to restore previous states
3. If data is truly missing, contact support immediately with details (what was there, when you last saw it)

---

## 🛠️ General Steps That Fix Most Issues

Try these first for any unexpected problem:

1. ✅ Refresh the page
2. ✅ Clear your browser cache
3. ✅ Try a different browser
4. ✅ Try incognito/private mode
5. ✅ Check your internet connection
6. ✅ Log out and log back in
7. ✅ Contact support if none of the above works

---

## 📞 Contact Support

If none of the steps above resolve your issue:

- Click **Help → Contact Support** from within the app
- Include: a description of the issue, steps to reproduce it, and a screenshot if possible
- Response time: **1–2 business days** for standard issues, **same day** for critical/blocking issues
